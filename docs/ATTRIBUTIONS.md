# Acknowledgements & Attributions

This project, **Futuristic Math Theorem Tester**, was made possible by leveraging the following incredible open-source projects, tools, and assets. We extend our sincere gratitude to their creators and maintainers.

## Core Technologies
- **[Python](https://www.python.org/)**: The programming language powering the backend engine.
- **[FastAPI](https://fastapi.tiangolo.com/)**: An elegant, high-performance web framework used to build our backend API and handle template rendering.
- **[Uvicorn](https://www.uvicorn.org/)**: A lightning-fast ASGI web server implementation.
- **[Jinja2](https://jinja.palletsprojects.com/)**: The modern and designer-friendly templating language for Python used to dynamically render our UI.

## Typography & Design Assets
- **[Google Fonts - Inter](https://fonts.google.com/specimen/Inter)**: The clean, highly legible typeface designed by Rasmus Andersson, which gives our user interface its sleek, modern aesthetic.
- **[shadcn/ui](https://ui.shadcn.com/)**: While the frontend primarily relies on custom vanilla CSS, the underlying structural inspiration and early layout concepts were heavily influenced by shadcn/ui. *(Used under the [MIT license](https://github.com/shadcn-ui/ui/blob/main/LICENSE.md))*
- **[Unsplash](https://unsplash.com)**: For conceptual placeholder imagery included in early Figma blueprints. *(Used under the [Unsplash License](https://unsplash.com/license))*

## Project Origins
- **[Figma](https://www.figma.com/)**: The original visual layout and neon-glassmorphism concepts were mocked up using Figma design tools.
